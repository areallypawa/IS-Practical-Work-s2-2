# Сравнение производительности двусвязного списка и динамического массива

Программа для изучения свойств двусвязных списков и динамических массивов, а также сравнения их производительности при выполнении основных операций.

## Цели работы

* Изучение свойств и организация двусвязных списков.
* Получение практических навыков в работе с динамическими массивами и двусвязными списками.
* Проведение сравнительной характеристики скорости:
  * вставки элементов;
  * получения элементов;
  * удаления элементов.

## Функциональность программы

Программа позволяет выполнять следующие действия для **двусвязного списка** и **динамического массива** (реализованы параллельно):

### 1. Формирование структуры данных

Пользователь может создать структуру двумя способами:

* **Автоматическое заполнение:** ввести количество элементов `N`, после чего структура заполняется случайными числами в диапазоне от 0 до 99.
* **Ручное заполнение:** вводить элементы в консоль поочерёдно, `N` определяется автоматически по количеству введённых элементов.

### 2. Измерение времени создания

Программа замеряет и выводит время, затраченное на создание структуры данных (двусвязного списка или динамического массива) выбранным способом.

### 3. Базовые операции

Для существующей структуры поддерживаются следующие операции:

* **Вставка** элемента в указанную позицию.
* **Удаление** элемента:
  * по индексу;
  * по значению (удаляется первый найденный элемент).
* **Получение** элемента по индексу.
* **Обмен** значений двух элементов по их индексам.

### 4. Измерение производительности операций

Для каждой из операций (вставка, удаление, получение) программа замеряет и отображает время их выполнения.

## Особенности реализации

* Возможность **многократного запуска** каждой операции в рамках одной сессии.
* Проверка на **пустую структуру**: если в списке/массиве нет элементов, программа выводит соответствующее предупреждение и не позволяет выполнить операции удаления или получения.
* **Сравнение производительности**: после выполнения операций для двусвязного списка и динамического массива выводятся результаты замеров для наглядного сравнения.

## Использование

1. Запустите программу.
2. Выберите структуру данных для работы: **двусвязный список** или **динамический массив**.
3. Создайте структуру данных одним из доступных способов (автоматически или вручную).
4. Выполняйте доступные операции (вставка, удаление, обмен, получение), наблюдая за временем их выполнения.
5. Повторите шаги 2–4 для второй структуры данных.
6. Сравните результаты замеров времени для обеих структур.
